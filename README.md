# Debita API

Rest API for [Debita App](https://debitapp.com) and [Debita Alexa Skill](https://debitapp.com)

## Installation

1. `npm i`

## Run offline

1. `npm run offline`

## Test
- Unit test
    - `npm run unit-test`
- Integration test
    - `npm run integration-test`
